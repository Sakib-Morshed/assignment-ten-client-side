# Habit Tracker Web Application

A full-stack MERN-based Habit Tracking website where users can create, update, manage, and track their daily habits to build streaks and boost productivity.

## Features

User Authentication: Users can register, login, and access private routes securely.
Add & Manage Habits: Users can add new habits with title, category, and details.
Personal Habit Dashboard:The “My Habits” page shows only the logged-in user's habits.Full CRUD System:Users can update, delete, or mark habits as complete.
Responsive UI: Beautifully responsive table layout for mobile, tablet, and desktop.
Latest Habits Section: Homepage shows the six most recently created habits.
Streak Tracking Logic:Automatically records and displays current streak days.
Real-Time Data: Data is stored and fetched from MongoDB Atlas using REST APIs.
